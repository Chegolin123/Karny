# 🎉 Karny — Telegram Mini App для планирования встреч

**Karny** — это Mini App внутри Telegram для создания комнат, планирования событий и общения с друзьями.

## ✨ Возможности

- 🔐 **Авторизация через Telegram** — вход по аккаунту Telegram
- 🏠 **Создание комнат** — приватные комнаты с уникальным кодом
- 📅 **Планирование событий** — создавай встречи и отмечай участие
- 💬 **Чат комнаты** — общайся с участниками в реальном времени
- 🌙 **Тёмная и светлая темы** — автоматическое переключение
- 📱 **Адаптивный дизайн** — работает на всех устройствах
- 🔔 **Уведомления** — напоминания о событиях через бота

## 🛠 Технологии

### Фронтенд
- React 18 + Vite
- Tailwind CSS
- React Router
- Telegram WebApp SDK
- WebSocket (чат)

### Бэкенд
- Node.js + Express
- MySQL (mysql2)
- WebSocket (ws)
- Telegram Bot API

### Инструменты
- Nodemon (автоперезагрузка)
- Serveo (туннель для разработки)
